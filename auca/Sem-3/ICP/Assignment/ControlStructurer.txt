3.Iteration Control Structure (Loops)
===========================================


-------------------------For Loop-------------------
i = 0
for i < 3:
    display("Hello", i)

----------------OUTPUT------------------

Hello 0
Hello 1
Hello 2


-------------------------while Loop------------------


count = 0
while count < 3:
    display("Count:", count)
    count += 1

----------------OUTPUT------------------

Count: 0
Count: 1
Count: 2

[Pre tested loop]

[ðŸŸ¨ Loops repeat code. You use them when you want to do something multiple times.]


------------------------- Do while Loop------------------


count = 0

do

    display("Count:", count)
    count += 1

    while count < 3:


----------------OUTPUT------------------

Count: 0
Count: 1
Count: 2

---------------------------------------------------------------------
do...while loop is a post-tested loop, meaning:

The code runs first, then the condition is checked.

It will always execute at least once, even if the condition is false.
